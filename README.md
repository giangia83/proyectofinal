# 🎉 Proyecto Final de **Gianfranco Iadarola** - **ED Técnica 2024**

## 📜 **Propósito**
El propósito de esta página web es servir como punto de encuentro online entre empresas y clientes que desean cotizar y comprar productos al por mayor, y el negocio **Starclean C.A.**.

## 🛍️ **¿Qué Vende?**
**Starclean C.A.** ofrece productos de:
- **Limpieza**
- **Seguridad Industrial**
- **Papelería**

## 💻 **Tecnologías Utilizadas**
La aplicación web cuenta con:
- **Storage de imágenes** vía BunnyCD (prueba gratuita)
- **Seguridad** con bcrypt para las contraseñas
- **Optimización WebP** para las imágenes
- **Sharp** para el recorte de imágenes
- **MongoDB** para la base de datos
- **Nodemailer** para enviar correos
-  Muchas noches sin dormir.
